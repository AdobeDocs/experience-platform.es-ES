# Monitorización de flujos de datos

[!UICONTROL Destinos] permite activar los datos de Adobe Experience Platform en innumerables socios externos.

Los flujos de datos son una representación de los trabajos de datos que mueven datos a través de [!DNL Platform]. Estos flujos de datos se configuran en diferentes servicios, lo que ayuda a mover datos de conectores de origen a conjuntos de datos de destino, donde luego los utiliza [!DNL Identity Service] y[!DNL Real-Time Customer Profile] antes de activarse finalmente en los destinos.

Para obtener más información sobre la monitorización de flujos de datos para destinos, lea la [Monitorización de flujos de datos para destinos en la IU](../../dataflows/ui/monitor-destinations.md) tutorial.
