# Recursos adicionales

>[!NOTE]
>
>Adobe Experience Platform Launch se está convirtiendo en un conjunto de tecnologías de recopilación de datos en Experience Platform. Como resultado, se han implementado varios cambios terminológicos en la documentación del producto. Consulte el siguiente [documento](../term-updates.md) para obtener una referencia consolidada de los cambios terminológicos.

La comunidad de etiquetas de Adobe Experience Platform está llena de profesionales con mucho talento que comparten consejos, tutoriales y herramientas útiles en la web. Esta es una recopilación de algunas de las mejores cosas que ha proporcionado la comunidad hasta ahora.

## Uso general

El repositorio [Awesome Adobe Launch](https://github.com/MisterPhilip/awesome-adobe-launch) GitHub proporciona una lista seleccionada por la comunidad de material relacionado con las etiquetas.
