# Creación de un archivo zip de paquete de extensiones

>[!NOTE]
>
>Adobe Experience Platform Launch se ha convertido en un grupo de tecnologías de recopilación de datos en Adobe Experience Platform. Como resultado, se han implementado varios cambios terminológicos en la documentación del producto. Consulte el siguiente [documento](../../term-updates.md) para obtener una referencia consolidada de los cambios terminológicos.

Para preparar un paquete de extensiones para cargarlo en Adobe Experience Platform, consulte los documentos de la API correspondientes a la herramienta de línea de comandos [@adobe/reactor-packager](https://www.npmjs.com/package/@adobe/reactor-packager). Genera un archivo zip preparado en el formato esperado para [cargar una extensión](./upload-and-test.md).
