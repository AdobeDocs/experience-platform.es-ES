# Tipos de acción

Esta página enumera todas las acciones admitidas dentro de una regla de etiqueta. Para crear o editar las acciones dentro de una regla de etiqueta:

1. Inicie sesión en [experience.adobe.com](https://experience.adobe.com) con sus credenciales de Adobe ID.
1. Vaya a **[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**.
1. Seleccione la propiedad de etiquetas que desee.
1. Vaya a **[!UICONTROL Rules]** y, a continuación, seleccione la regla que desee.
1. En [!UICONTROL Actions], seleccione una acción existente o cree una acción.
1. Establezca el campo desplegable [!UICONTROL Extension] en **[!UICONTROL Adobe Experience Platform Web SDK]** y, a continuación, seleccione el [!UICONTROL Action Type] que desee.

La extensión de etiquetas Web SDK ofrece las siguientes acciones:

* [**[!UICONTROL Apply propositions]**](apply-propositions.md): procesar propuestas en aplicaciones de una sola página sin incrementar las métricas.
* [**[!UICONTROL Apply response]**](apply-response.md): realice una acción basada en una respuesta de Edge Network.
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md): Almacene en déclencheur manualmente una evaluación del conjunto de reglas.
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md): exporte la API de medios heredada a un objeto window.
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md): permite compartir un identificador de visitante entre los dominios de su propiedad.
* [**[!UICONTROL Send event]**](send-event.md): envía datos de evento a Edge Network.
* [**[!UICONTROL Send media event]**](send-media-event.md): envía datos de medios a Edge Network.
* [**[!UICONTROL Set consent]**](set-consent.md): establece el consentimiento deseado para el visitante.
* [**[!UICONTROL Update variable]**](update-variable.md): modifica un elemento de datos [variable](../data-element-types.md#variable).
