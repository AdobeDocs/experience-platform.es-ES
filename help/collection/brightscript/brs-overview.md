# Información general de Brightscript

El **SDK Roku de Adobe Experience Platform** le permite enviar datos a un [conjunto de datos](/help/datastreams/overview.md) desde un dispositivo Roku usando Brightscript.

SDK y la documentación de esta biblioteca están alojados en GitHub:

**[Adobe Experience Platform Roku SDK](https://github.com/adobe/aepsdk-roku)**
