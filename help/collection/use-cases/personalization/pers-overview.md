# Información general sobre casos de uso de Personalization

Adobe Experience Platform Web SDK permite a las empresas proporcionar una amplia variedad de casos de uso de personalización a sus propiedades web. Puede conectar sin problemas los procesos de recopilación, toma de decisiones y activación de datos en soluciones de Adobe como Target, Journey Optimizer y Audience Manager.

SDK admite arquitecturas flexibles, incluidos modelos del lado del cliente, del lado del servidor e híbridos, lo que permite recuperar y procesar ofertas personalizadas que cumplan con la mayoría de los casos de uso de la organización.

## Ejemplos de Web SDK

Además de las páginas de documento de esta carpeta, Adobe mantiene un repositorio de aplicaciones de ejemplo a las que puede hacer referencia. Consulte [Ejemplos de Web SDK](https://github.com/adobe/alloy-samples/) en GitHub para ver más casos de uso de personalización, entre los que se incluyen:

* Personalización del lado del cliente
* Personalización del lado del servidor
* Personalización híbrida
* Personalization con aplicaciones de una sola página
