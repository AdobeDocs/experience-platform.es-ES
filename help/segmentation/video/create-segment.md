# Creación de segmentos en la interfaz de usuario de Experience Platform

El siguiente vídeo pretende ayudarle a comprender la creación de segmentos. Muestra un ejemplo de experiencia del cliente seguido de un recorrido rápido por las características clave de la interfaz [!DNL Platform]. Para obtener más información acerca de la segmentación y cómo crear segmentos en la interfaz de usuario, lea la [guía del Generador de segmentos](../ui/segment-builder.md).

Si está interesado en crear un segmento usando la API de segmentación, lea [crear una guía de API de segmento](../tutorials/create-a-segment.md).

>[!NOTE]
>
>La interfaz de usuario del Experience Platform se actualiza con frecuencia y puede haber cambiado desde que se grabó este vídeo. Para obtener la información más actualizada, consulte la [guía del Generador de segmentos](../ui/segment-builder.md).
