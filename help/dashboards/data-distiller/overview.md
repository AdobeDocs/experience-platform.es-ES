# Resumen de Data Distiller

Data Distiller es un paquete de ofertas que incluye un subconjunto de las funcionalidades de Adobe Experience Platform. Con Data Distiller puede realizar la preparación de datos posterior a la ingesta (como limpieza, forma y manipulación) para casos de uso analítico o del perfil del cliente en tiempo real ejecutando consultas por lotes en Query Service. También puede [crear modelos de datos de informes personalizados](../../query-service/data-distiller/sql-insights/reporting-insights-data-model.md) o [usar SQL para generar gráficos para sus paneles personalizados](./sql-insights/overview.md).

>[!NOTE]
>
>El uso de Data Distiller depende de los derechos que tenga para las aplicaciones basadas en Platform.
