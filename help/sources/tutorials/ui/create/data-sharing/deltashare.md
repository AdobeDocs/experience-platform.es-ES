# Usar el conector de origen [!DNL Deltashare] en la interfaz de usuario {#use-deltashare-in-the-ui}

>[!CONTEXTUALHELP]
>id="platform_sources_deltashare_schema"
>title="Estructura del esquema"
>abstract="Asegúrese de revisar la estructura de esquema, ya que una vez que continúe, ya no podrá realizar cambios en el esquema."

Lea esta guía para aprender a utilizar el conector de origen [!DNL Deltashare] en la interfaz de usuario de Adobe Experience Platform.

**Nota:** Una vez que seleccione **Siguiente**, no podrá cambiar la estructura de esquema seleccionada.

Si ya ha seleccionado **Siguiente** y ha pasado el paso de selección de esquema, ya no podrá actualizar el esquema seleccionado si vuelve al paso anterior. Para modificar el esquema, debe reiniciar el proceso de configuración del flujo de datos y comenzar desde el paso inicial.
