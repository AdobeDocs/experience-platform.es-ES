# LISTA DE PERMITIDOS de direcciones IP

Puede definir controles de acceso de red a través del servidor de seguridad de red. Al especificar el rango de IP de origen adecuado, puede permitir el tráfico para el servicio de transferencia de datos. Las siguientes direcciones IP deben agregarse a una lista de permitidos antes de trabajar con conectores de origen. Si no se agregan las direcciones IP específicas de su región a la lista de permitidos, puede que se produzcan errores o no se produzca un rendimiento al usar fuentes.
