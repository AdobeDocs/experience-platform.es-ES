# Conectores Azure Blob y Amazon S3

Adobe Experience Platform proporciona conectividad nativa para proveedores de nube como AWS, [!DNL Google Cloud Platform]y [!DNL Azure]. Puede incorporar los datos de estos sistemas a [!DNL Platform].

Las fuentes de almacenamiento en la nube pueden incorporar sus propios datos a [!DNL Platform] sin necesidad de descargar, formatear o cargar. Los datos introducidos pueden tener el formato XDM JSON, XDM Parquet o delimitados. Cada paso del proceso se integra en el flujo de trabajo Orígenes . [!DNL Platform] permite introducir datos de [!DNL Azure Blob] y S3 por lotes.

Para obtener más información sobre [!DNL Azure Blob] conector, lea la [[!DNL Azure Blob] información general del conector](./blob.md).

Para obtener más información sobre [!DNL Amazon S3] conector, lea la [[!DNL Amazon S3] información general del conector](./s3.md).
