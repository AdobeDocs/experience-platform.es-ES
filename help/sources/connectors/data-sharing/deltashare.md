# [!DNL Deltashare]

Puede usar el conector de origen [!DNL Deltashare] para establecer una conectividad segura con [!DNL Databricks Delta Sharing] servidores y crear conjuntos de datos virtuales en Adobe Experience Platform. Con esta función, puede consultar conjuntos de datos virtuales, inspeccionar y asignar tablas externas a esquemas XDM (Experience Data Model) y aprovecharlos en aplicaciones y servicios de Experience Platform. A través de [!DNL Deltashare], puede evitar la ingesta física de conjuntos de datos, reduciendo así la duplicación, los costos de almacenamiento y la sobrecarga operativa.
