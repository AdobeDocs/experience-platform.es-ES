# Análisis y seguimiento del consentimiento

En el panorama de marketing actual, debe comprender y respetar las preferencias de consentimiento del cliente. Adobe Real-time Customer Data Platform permite a los especialistas en marketing analizar el consentimiento de los clientes para generar confianza, cumplir con las normas de privacidad y ofrecer experiencias más personalizadas.

La guía de análisis de consentimiento detalla cómo crear un panel de consentimiento para varios casos de uso de marketing para datos de Real-Time CDP. Para obtener información sobre cómo crear un panel de análisis de consentimiento con el fin de realizar un seguimiento de las tendencias del consentimiento del usuario a lo largo del tiempo, consulte la [guía del panel de análisis de consentimiento](../../dashboards/insights-use-cases/consent-analysis.md).
