# Paquete de Data Distiller

Data Distiller es un paquete de ofertas que incluye un subconjunto de las funcionalidades de Adobe Experience Platform. Este paquete está diseñado para la preparación de datos posteriores a la ingesta para casos de uso de datos analíticos o perfiles de clientes en tiempo real mediante consultas por lotes ejecutadas en el servicio de consultas. Consulte la [Resumen de Data Distiller](../data-distiller/overview.md) para obtener más información sobre el paquete de Data Distiller.
