# Paquete de Distiller de datos

Data Distiller es una oferta de paquetes que incluye un subconjunto de las funcionalidades de Adobe Experience Platform. Este paquete está diseñado para la preparación de datos posteriores a la ingesta para perfiles de clientes en tiempo real o casos de uso de datos analíticos mediante consultas por lotes ejecutadas en el servicio de consultas. Consulte la [Información general de Data Distiller](../data-distiller/overview.md) para obtener más información sobre el paquete de Data Distiller.
