# Sugerencias principales para maximizar el valor con Adobe Experience Platform Data Distiller

Esta página contiene el conjunto de datos de ejemplo para que aplique lo que ha aprendido en la sesión &quot;OS656: consejos principales para maximizar el valor con Adobe Experience Platform Data Distiller&quot;. Aprenderá a acelerar las implementaciones de Adobe Real-time Customer Data Platform y Journey Optimizer enriqueciendo los datos del perfil del cliente en tiempo real. Este enriquecimiento aprovecha la perspectiva profunda de los patrones de comportamiento de los clientes para crear audiencias para la entrega y optimización de experiencias.

A través de un caso práctico, analizará los datos de comportamiento del usuario y creará un modelo de actualización, frecuencia y monetario (RFM), una técnica de análisis de marketing para la segmentación de clientes basada en patrones de compra.
