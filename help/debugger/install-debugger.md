# Instalar Experience Platform Debugger

Adobe Experience Platform Debugger se puede instalar en el explorador Chrome de Google. Siga el vínculo a continuación para instalar la extensión en su explorador:

* [Chrome](https://chrome.google.com/webstore/detail/adobe-experience-platform/bfnnokhpnncpkdmbokanobigaccjkpob)

Después de instalar la extensión de Chrome, se agrega un icono (![](images/start-icon.jpg)) a la barra de extensiones. Seleccione este icono para abrir la extensión de.
