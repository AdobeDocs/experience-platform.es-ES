# Creación de una conexión de flujo continuo mediante la API

Se requiere el registro de la conexión de flujo continuo para iniciar la transmisión de datos a Adobe Experience Platform. Al registrar una conexión de flujo continuo, debe proporcionar algunos detalles clave como el origen de los datos de flujo continuo.

Después de registrar una conexión de flujo continuo, usted, como productor de datos, tendrá una URL única que se puede utilizar para transmitir datos a Experience Platform.

Para obtener información sobre cómo crear una conexión de flujo continuo autenticada, lea la [guía de la API de origen HTTP API](../../sources/tutorials/api/create/streaming/http.md).
