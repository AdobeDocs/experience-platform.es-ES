# Fuentes en la plataforma de datos del cliente en tiempo real B2B Edition

La plataforma de datos del cliente en tiempo real B2B Edition le permite introducir sus datos B2B a través del conector de origen [[!DNL Marketo Engage]](../../sources/connectors/adobe-applications/marketo/marketo.md), al tiempo que le ofrece la capacidad de estructurar, etiquetar y mejorar los datos entrantes mediante los servicios.

Los siguientes documentos proporcionan más información sobre el uso de [!DNL Marketo] para traer datos B2B a Platform:

* Para obtener información sobre cómo conectar los datos [!DNL Marketo] a Platform, consulte el tutorial sobre la [creación de un conector de origen de Marketo en la interfaz de usuario](../../sources/tutorials/ui/create/adobe-applications/marketo.md).
* Para obtener información sobre la configuración subyacente de los espacios de nombres y esquemas B2B utilizados con [!DNL Marketo], consulte la documentación de [Espacios de nombres B2B y esquemas](../../sources/connectors/adobe-applications/marketo/marketo-namespaces.md).
* Para obtener información sobre cómo encontrar su [!DNL Marketo] ID de munchkin y generar sus credenciales, consulte la [[!DNL Marketo] guía de autenticación](../../sources/connectors/adobe-applications/marketo/marketo-auth.md).
* Para obtener información sobre las reglas de asignación específicas que se aplican a los conjuntos de datos [!DNL Marketo], consulte la documentación sobre [[!DNL Marketo] asignaciones de campo](../../sources/connectors/adobe-applications//mapping/marketo.md).
